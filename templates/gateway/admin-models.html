На этой странице не подключается внешний API автоматически по API-ключу, потому что внешние провайдеры не имеют единого стандарта получения лимитов/цен через один endpoint. Поэтому сейчас оставлен безопасный ручной режим: редактировать цену и активность существующих моделей.
